# CodeTED

> Soluciones digitales que impulsan negocios.

---

# Visión

CodeTED es una plataforma SaaS desarrollada para ofrecer soluciones tecnológicas profesionales a pequeñas y medianas empresas.

No es únicamente una página web corporativa.

Es una plataforma completa que permitirá administrar clientes, proyectos, cotizaciones, contenido, automatizaciones y servicios desde un único lugar.

El proyecto servirá además como portafolio profesional demostrando buenas prácticas de desarrollo Full Stack utilizando tecnologías modernas.
